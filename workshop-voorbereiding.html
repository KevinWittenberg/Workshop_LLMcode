Voorbereiding workshop: Code schrijven met ChatGPT

Gebruik deze handout om vóór de workshop je accounts klaar te zetten. Je hebt een GitHub-account nodig, een ChatGPT-account met je @scp.nl-werkmail, en een koppeling tussen ChatGPT en GitHub.

Wat moet klaar zijn?

  • Je kunt inloggen op github.com.
  • Je kunt inloggen op chatgpt.com met je @scp.nl-adres.
  • In ChatGPT staat GitHub gekoppeld via Settings → Apps.
  • Je hebt alleen repositories geselecteerd die je voor de workshop nodig hebt.

1. Maak een GitHub-account

  1. Ga naar github.com.
  2. Klik rechtsboven op Sign up / Aanmelden.
  3. Vul je e-mailadres in en volg de stappen op het scherm.
  4. Bevestig je e-mailadres via de mail die GitHub stuurt.
  5. Kies een sterk wachtwoord. Gebruik bij voorkeur een wachtwoordmanager.

2. Maak je eigen kopie van de workshop-repository op GitHub

Voor deze workshop hoef je Git niet lokaal te installeren. De eenvoudigste route is om de repository rechtstreeks op GitHub te forken. Een fork is je eigen GitHub-kopie van de oorspronkelijke repository. Je kunt daarna met jouw fork werken, zonder dat je eerst iets hoeft te downloaden.

  1. Log in op GitHub.
  2. Ga naar github.com/KevinWittenberg/Workshop_LLMcode.
  3. Klik rechtsboven op Fork.
  4. Laat de naam bij voorkeur staan als Workshop_LLMcode.
  5. Kies je eigen GitHub-account als eigenaar.
  6. Klik op Create fork.
  7. Controleer dat je nu op een adres zit zoals https://github.com/jouw-gebruikersnaam/Workshop_LLMcode. Dat is de repository die je straks gaat gebruiken.
Wat de **** is een fork? Daarmee staat de workshopcode direct in je eigen GitHub-omgeving. Je hoeft niets lokaal te downloaden. GitHub beschouwt een fork als een nieuwe repository die een kopie is van het oorspronkelijke project, en daarmee dus ook automatisch kan updaten als de originele bestanden wijzigen / updaten.

3. Maak of open je ChatGPT-account met je @scp.nl-adres

  1. Ga naar chatgpt.com.
  2. Klik op Sign up als je nog geen account hebt. Heb je al een account met je werkmail, kies dan Log in.
  3. Gebruik je @scp.nl-werkmail. Gebruik geen privé-account voor deze workshop. Dat staat in de richtlijnen van het SCP en voorkomt dat herinneringen of context uit privégesprekken en werkgesprekken door elkaar gaan lopen.
  4. Doorloop de verificatie in je mailbox.
Tip: zie je binnen ChatGPT meerdere accounts of environments? Kies de omgeving die bij je SCP-werkmail hoort voordat je GitHub koppelt.

4. Koppel GitHub aan ChatGPT en voeg je workshop-repository toe aan Codex

Ga in ChatGPT naar Settings → Apps, zoek GitHub, en laat je daarna doorsturen naar GitHub om de ChatGPT-app te installeren en repositories te kiezen.

  1. Zorg dat je chatgpt.com open hebt.
  2. Klik linksonder of rechtsboven op je profiel/naam en open Settings.
  3. Ga naar Apps. In sommige omgevingen heet dit Apps & Connectors of Connectors.
  4. Zoek GitHub en klik op Connect.
  5. Klik op Continue to GitHub als ChatGPT daarom vraagt.
  6. Log in bij GitHub, keur de koppeling goed en kies de repositories die ChatGPT mag gebruiken.
  7. Selecteer expliciet je eigen fork: jouw-gebruikersnaam/Workshop_LLMcode.
  8. Wacht enkele minuten als je repository niet meteen zichtbaar is. Volgens OpenAI kan er tot ongeveer vijf minuten vertraging zijn.

5. Controleer of alles werkt

  1. Open ChatGPT.
  2. Ga opnieuw naar Settings → Apps.
  3. Open GitHub. De status moet aangeven dat GitHub gekoppeld is.
  4. Zie je je repository niet meteen? Wacht een paar minuten en vernieuw ChatGPT.
Let op: de GitHub-app is niet in elk ChatGPT-plan of elke workspace op precies dezelfde manier zichtbaar. Als je GitHub niet ziet, controleer dan eerst of je in de juiste SCP-werkruimte zit. Lukt er iets niet? Kom dan iets eerder voor de workshop langs zodat we je kunnen helpen.

Privacy en veilig werken

  • Koppel alleen de repository die je nodig hebt voor de workshop.
  • Gebruik bij voorkeur je eigen fork: jouw-gebruikersnaam/Workshop_LLMcode.
  • Gebruik geen echte onderzoeksdata, persoonsgegevens, geheime sleutels of vertrouwelijke projectinformatie.
  • Twijfel je of iets gedeeld mag worden? Deel het dan niet en vraag eerst om hulp.
